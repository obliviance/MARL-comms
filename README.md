# MARL-comms

## Setup
1. Create a python environment using `python -m venv env`.
2. Start the environment.
    On Windows `source env/Scripts/activate`
    On Linux `source env/bin/activate`
3. Install the dependencies with `pip install -r requirements.txt`


## Environment
The main environment we are investigating is the Multi Particle Environment(MPE) available through PettingZoo, which is a python library associated with Farama Gymnasium, that provides multiagent environments.

We may also investigate the Drone Swarm Search environment.
